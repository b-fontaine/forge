## Implementer Notes (carried from design.md Anti-Hallucination Pass)

1. **`pin_review_cadence:` values use ISO 8601 (`P30D` / `P12M`), NOT
   `30d` / `12mo`.** ADR-B8-SIG-005 supersedes the informal notation in
   proposal Solution #3 + spec FR-B8-SIG-B-005. The L1 cadence-present
   regex (FR-B8-SIG-E-010) MUST be authored against the ISO 8601 form.
2. **WAIVER block + CHANGELOG + `infra/CLAUDE.md.tmpl` cite
   `ADR-J7-004`** (the actual schema-relaxation precedent —
   `additionalProperties: true` at root, `j7-validate-standards-yaml/design.md`
   lines 152-168), **NOT `ADR-J7-008`** (proposal § Article XII + spec
   FR-B8-SIG-B-009 mis-citation). Per Article V the frozen proposal/specs
   are NOT edited ; the correction lands only in the WAIVER + CHANGELOG +
   doc per ADR-B8-SIG-005 Consequences.

   containers), NOT 2 or 3.** ADR-B8-SIG-001 + ADR-B8-SIG-007 supersede
   the proposal's "2-service vs 3-service" framing. The 4 long-running :
   `fsm-signoz` (`signoz/signoz:v0.125.1`), `fsm-signoz-otel-collector`
   (`signoz/signoz-otel-collector:v0.144.4`), `fsm-signoz-clickhouse`
   (`clickhouse/clickhouse-server:25.5.6`), `fsm-signoz-zookeeper`
   (`signoz/zookeeper:3.7.1`). The 2 init : `init-clickhouse`
   (`clickhouse/clickhouse-server:25.5.6`) +
   `fsm-signoz-telemetrystore-migrator`
   (`signoz/signoz-otel-collector:v0.144.4`).
